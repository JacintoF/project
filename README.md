# Navigation Final Project For LMC 3226
## By: Jacinto Fernandez and Nathan Banks
This website was custom coded to show how time and space has been used to orient people across time. 
Furthermore, it shows how time and space can be bent to alter people's perception of where they are and how long they have been there.
